## 🎯 How to Display John 3:16

### Method 1: Click Method
```
1. Book dropdown → Select "John"
2. Chapter input → Type "3"
3. Verse Start → Type "16"
4. Verse End → Type "16"
5. Click "Show on Display"
6. ✅ Done!
```

### Method 2: Keyboard Shortcuts
```
1. Tab to Book → Arrow keys to select "John"
2. Tab to Chapter → Type "3"
3. Tab to Verse → Type "16"
4. Press Enter anywhere
5. ✅ Done!
```

## 🔧 Troubleshooting

### Problem: No verses appear
**Solution:** Make sure you selected John 3 or Psalms 23

### Problem: Red errors in console
**Solution:** Refresh page (Ctrl+Shift+R)

### Problem: Display window blank
**Solution:** 
1. Click "Open Display" in header
2. Make sure display window is visible
3. Click "Show on Display" again

### Problem: Keyboard shortcuts don't work
**Solution:** Click somewhere outside input fields first
