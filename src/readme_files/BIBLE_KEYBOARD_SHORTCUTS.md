# ⌨️ Bible Display Keyboard Shortcuts - Quick Reference

## 🎯 Main Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Enter** | Show on Display | Opens/updates presentation window on projector |
| **P** | Preview Only | Shows verse in preview panel (not on projector) |
| **Esc** | Clear Display | Clears all displays and closes windows |

## 📖 Navigation Shortcuts

| Shortcut | Action | Auto-Update |
|----------|--------|-------------|
| **←** | Previous Verse | ✅ Yes - if display open |
| **→** | Next Verse | ✅ Yes - if display open |
| **↑** | Previous Chapter | ⚠️ No - press Enter to display |
| **↓** | Next Chapter | ⚠️ No - press Enter to display |
| **PgUp** | Previous Book | ⚠️ No - press Enter to display |
| **PgDn** | Next Book | ⚠️ No - press Enter to display |

## 💡 Auto-Update Feature

When the Bible presentation window is **already open**:

- Press **←** or **→** to navigate verses
- The display **automatically updates** - no need to press Enter again!
- Perfect for quickly showing multiple verses during a sermon
- **NEW:** Change ANY selection (book, chapter, verse) and it updates automatically!
- Updates happen within 300ms of changing the selection

Example workflow:
1. Press **Enter** once (opens presentation window)
2. Drag window to projector, press F11
3. During sermon, just change verse numbers by typing
4. Display updates automatically! ⚡
5. Change chapter - auto-updates! ⚡
6. Change book - auto-updates! ⚡

## 🎮 Typical Church Service Usage

### Opening Sermon Scripture:

```
1. Select: John 3:16
2. Press: Enter
3. Drag window to projector
4. Press: F11 (fullscreen)
✅ Congregation sees verse!
```

### Showing Multiple Related Verses:

```
1. Display is already open on John 3:16
2. Press: → (moves to verse 17)
3. Press: → (moves to verse 18)
4. Press: → (moves to verse 19)
✅ All verses auto-update on projector!
```

### Jumping to Different Chapter:

```
1. Currently on John 3:16
2. Press: ↓ (changes to chapter 4)
3. Press: Enter (updates display)
✅ Now showing John 4:16
```

### Checking Before Showing:

```
1. Select: Psalms 23:1
2. Press: P (preview only)
3. Check preview panel
4. Press: Enter (show on projector)
✅ Verified before projecting!
```

## ⚠️ Important Notes

### Shortcuts Work Globally
- Keyboard shortcuts work **anywhere** in the Bible tab
- Even when typing in input fields, **Enter** shows the verse
- Only exception: **P** only works when NOT typing

### Auto-Update Only for Verses
- **←/→** (verses): Auto-updates ✅
- **↑/↓** (chapters): Need to press Enter ⚠️
- **PgUp/PgDn** (books): Need to press Enter ⚠️

This design prevents accidental chapter/book changes during live services!

## 🎨 Display Window Behavior

### First Time Pressing Enter:
- Opens a new popup window
- Title: "Bible Display - John 3:16"
- Size: 1920x1080 (Full HD)
- You can drag it anywhere

### Pressing Enter Again:
- **Closes** the old window
- **Opens** a new window with updated verse
- This ensures clean updates

### Arrow Keys (←/→):
- **Updates** the existing window content
- Doesn't close and reopen
- Smooth and fast! ⚡

## 🚀 Pro Tips for Worship Leaders

### Tip 1: Pre-load Verses
Before service:
1. Open Bible tab
2. Navigate to your first verse
3. Press **Enter** to open display window
4. Minimize window (keep it open)
5. During service, just select verses and press Enter

### Tip 2: Rapid Verse Switching
For rapid verse display:
1. Open display once with Enter
2. Use **arrow keys only** to switch verses
3. Much faster than clicking buttons!

### Tip 3: Preview First
When learning new passages:
1. Press **P** to preview
2. Read it in preview panel
3. Press **Enter** when ready to show
4. Prevents mistakes!

### Tip 4: Use Both Hands
Left hand on arrow keys, right hand on Enter:
- Navigate with left (←→↑↓)
- Display with right (Enter)
- Maximum efficiency! ⚡

## 📱 Mobile/Tablet Note

Keyboard shortcuts only work on **desktop/laptop** with a physical keyboard.

For touch devices, use the buttons:
- "Show on Preview" button
- "Show on Display" button
- "Clear All" button

## 🎓 Training Your Team

To train volunteers:

1. **Show them this cheat sheet**
2. **Practice sequence:**
   - Select John 3:16
   - Press Enter (explain: "opens on projector")
   - Press → (explain: "next verse, auto-updates")
   - Press ← (explain: "previous verse")
   - Press Esc (explain: "clear everything")

3. **Common mistakes:**
   - Forgetting to press Enter after changing chapters
   - Pressing P when they meant Enter
   - Not knowing about auto-update feature

4. **Best practices:**
   - Keep display window open for the whole service
   - Use preview first if unsure
   - Arrow keys are your friend!
