# Quick Reference Card - Display Operator

## Before Service Starts

### Setup (5 minutes)
1. ✅ Connect projector to computer
2. ✅ Set display mode to **"Extend"** (Windows + P → Extend)
3. ✅ Login to app (admin or user account)
4. ✅ Click **"Open Display"** button (top header)
5. ✅ Drag display window to projector screen
6. ✅ Press **F11** for fullscreen
7. ✅ Test background visibility

### Admin Background Setup (Optional)
1. Click **"Admin Panel"**
2. Go to **"Display Settings"** tab
3. Set background color or image URL
4. Verify in preview

---

## During Service

### Displaying Bible Verses

```
1. Click "Bible" tab
2. Select: Version → Book → Chapter → Verses
3. Press ENTER (or click "Show on Display")
4. Verse appears on projector!
```

**Navigate verses:**
- `←/→` keys: Previous/Next verse
- `↑/↓` keys: Previous/Next chapter  
- `Esc`: Clear display

### Displaying Songs

```
1. Click "Songs" tab
2. Search or browse for song
3. Click song to open viewer
4. Click "Show" button (eye icon)
5. Song appears on projector!
```

**Navigate slides:**
- `←/→` keys: Previous/Next slide
- `Esc`: Close viewer (clears display)

---

## Keyboard Shortcuts Cheat Sheet

| Key | Action |
|-----|--------|
| `Enter` | Show on Display |
| `←` | Previous (verse/slide) |
| `→` | Next (verse/slide) |
| `↑` | Previous chapter |
| `↓` | Next chapter |
| `Esc` | Clear/Close |
| `F11` | Fullscreen (on display window) |

## Common Tasks

### Switch Between Bible and Songs
- Just click the **Bible** or **Songs** tab
- Display window stays open
- No need to reopen

### Change Background Mid-Service
1. Admin Panel → Display Settings
2. Choose new color/image
3. Apply (changes immediately)

### Multiple Verses Display
1. Enter start verse (e.g., 16)
2. Enter end verse (e.g., 18)
3. Shows "John 3:16-18" with all verses

### Clear Everything Quickly
- Press `Esc` on keyboard
- Display shows idle wallpaper

## Troubleshooting

| Problem | Quick Fix |
|---------|-----------|
| Display not updating | Refresh both windows |
| Can't see display window | Check if minimized or on wrong screen |
| Background not showing | Verify image URL works in browser |
| Keyboard shortcuts not working | Click on the control window first |
| Display window closed | Click "Open Display" again |

- ✅ **Esc to clear** between items
- ✅ **Test before service** starts

---

## Emergency Procedures

**If display freezes:**
1. Press `Esc` to clear
2. If still frozen, refresh display window (F5)
3. If needed, close and click "Open Display" again

**If wrong content showing:**
1. Press `Esc` immediately
2. Reselect correct content
3. Press `Enter` to display

**If lost connection:**
1. Check internet connection
2. Refresh both windows
3. Reselect content
