# 🚀 QUICK START - Kebena Church Display App

## ⚡ **Start Using in 2 Minutes**

### **1. Login**
- Username: `admin` 
- Password: (your admin password)

### **2. Setup Projector Display**
1. Click **"Open Display"** button (top right)
2. Drag new window to your projector screen
3. Press **F11** for fullscreen

### **3. Display Bible Verse**
1. Click **"Bible"** tab
2. Select: Book → Chapter → Verse
3. Press **`Enter`** key
4. **Verse appears on projector!**

### **4. Navigate with Keyboard**
- `→` = Next verse
- `←` = Previous verse  
- `Esc` = Clear display

---

```

↓ = Next chapter

```

## 🎨 **Change Background (Admin Only)**

### **Option 1: Quick Preset Images**
1. Click **"Admin Panel"**
2. Go to **"Settings"** tab
3. Click any preset:
   - 🏛️ Church Interior
   - ✝️ Cross Background
   - 📖 Bible Background

### **Option 2: Custom Image URL**
1. Find image URL (Unsplash, Pexels, etc.)
2. Paste in "Image URL" field
3. Press **Enter**

**Example URL to try:**
```
https://images.unsplash.com/photo-1438232992991-995b7058bbb3?w=1920&q=80
```

---

## 🎵 **Display Songs**

1. Click **"Songs"** tab
2. Search or browse songs
3. Click on song
4. Use **`→`** and **`←`** to navigate slides
5. Press **`Esc`** to close

## 🐛 **Quick Troubleshooting**

**Display not showing on projector?**
- Set Windows display to "Extend" (Win+P)
- Drag window to second screen manually
- Press F11 for fullscreen

**Bible not loading?**
- Check internet connection
- Try John 3:16 (works offline)
- Refresh page

**Background not showing?**
- Verify URL is valid (must be HTTPS)
- Try a preset background first
- Check image is accessible
