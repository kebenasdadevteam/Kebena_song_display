# 📖 Bible Feature - Quick Reference Card

## 🚀 5-Second Setup
1. Click **"Open Display"** button (top right)
2. Press **F11** on new window
3. Move to projector
4. Click **"Bible"** tab in main window
5. Start displaying verses!

## 🎯 Display a Verse (4 Steps)
```
1. Select Version + Book + Chapter + Verse
2. Check preview (right panel)
3. Click "Show on Display"
4. Click "Clear Display" when done
```

## 📍 Where Everything Is

### Main App Window
- **Songs Tab**: Your existing song display (unchanged)
- **Bible Tab**: New Bible control panel
- **Open Display Button**: Top right in header (Monitor icon)

### Display Window
- **Full Screen**: Press F11
- **Shows**: Only what you choose to display
- **Updates**: Instantly when you click "Show"

## 🔑 Key Buttons

| Button | What It Does |
|--------|-------------|
| **Open Display** | Opens projector window |
| **Show on Display** | Displays selected verse |
| **Clear Display** | Shows church wallpaper |
| **Songs Tab** | Access song display |
| **Bible Tab** | Access Bible display |

## ⚡ During Service Workflow

### For Songs
```
Songs Tab → Select Song → Display
(Everything works as before)
```

### For Bible Verses
```
Bible Tab → Select Verse → Preview → Show → Clear
```

### Switching Between Them
```
Just click the tab you need!
No need to close anything.
```

## 💡 Pro Tips

✅ **Always preview** before showing
✅ **Clear display** after each verse
✅ **Keep display window open** during entire service
✅ **Searching is private** - congregation doesn't see your selections

## 🔄 Real-Time Sync

**Control Screen** → **Supabase Cloud** → **Display Screen(s)**

- Updates in < 100ms
- Works across devices
- Requires internet

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Display not updating | Refresh display window |
| Can't see preview | Open display window first |
| Wrong verse showing | Double-check selections |
| Placeholder text | Sample data (need XML files) |

## 📱 Two Screen Modes

### Control Mode (Your PC)
**URL**: Normal app URL
**Shows**: Full app with tabs and controls
**For**: Operators

### Display Mode (Projector)
**URL**: Add `?mode=display` to URL
**Shows**: Full screen content only
**For**: Congregation

## ⚙️ Settings

**Admin Panel → Settings → Presentation Background**
- Changes background for both songs and Bible
- Default: Dark blue (#1a1a2e)

## 🔐 Permissions

**All Users**: Can display Bible verses
**Admins**: Can also change settings
