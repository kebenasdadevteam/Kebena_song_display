## ⌨️ **KEYBOARD SHORTCUTS**

### **BIBLE CONTROL** 📖
```
┌─────────────────────────────────────────┐
│  Enter    →  Display verse on projector │
│  P        →  Preview only (no projector)│
│  Esc      →  Clear all displays         │
│  →        →  Next verse (auto-update)   │
│  ←        →  Previous verse             │
│  ↓        →  Next chapter               │
│  ↑        →  Previous chapter           │
│  PgDn     →  Next book                  │
│  PgUp     →  Previous book              │
└─────────────────────────────────────────┘
```

### **SONG CONTROL** 🎵
```
┌─────────────────────────────────────────┐
│  →        →  Next slide                 │
│  ←        →  Previous slide             │
│  Home     →  First slide                │
│  End      →  Last slide                 │
│  Esc      →  Close song viewer          │
└─────────────────────────────────────────┘
```

### **DISPLAY WINDOW** 🖥️
```
┌─────────────────────────────────────────┐
│  F11      →  Fullscreen toggle          │
│  Esc      →  Clear display              │
└─────────────────────────────────────────┘
```

## 🎯 **QUICK WORKFLOWS**

### **Display Bible Verse**
```
1. Click "Bible" tab
2. Select: Book → Chapter → Verse
3. Press Enter
4. Use → to navigate
5. Press Esc when done
```

### **Display Song**
```
1. Click "Songs" tab
2. Click song from list
3. Use → to advance slides
4. Press Esc when done
```

### **Setup Projector**
```
1. Click "Open Display"
2. Drag window to projector
3. Press F11 for fullscreen
```

## 🐛 **QUICK FIXES**

### **Display not on projector?**
```
✓ Windows: Press Win+P → Select "Extend"
✓ Mac: System Preferences → Displays
✓ Drag display window manually
```

### **Bible not loading?**
```
✓ Check internet connection
✓ Try John 3:16 (works offline)
✓ Refresh page
```

### **Background not showing?**
```
✓ URL must start with https://
✓ Try a preset first
✓ Check image is accessible
```
