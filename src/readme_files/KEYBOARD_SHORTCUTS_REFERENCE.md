# ⌨️ Keyboard Shortcuts Reference - Kebena Church Display

## 📖 **BIBLE CONTROL** - All Shortcuts Working ✅

| Key | Action | Details |
|-----|--------|---------|
| **`Enter`** | **Display Verse** | Shows selected verse on projector window |
| **`P`** | **Preview Only** | Shows verse in preview panel (not on projector) |
| **`Esc`** | **Clear Display** | Removes all content from display & closes windows |
| **`←`** (Left) | **Previous Verse** | Goes back one verse, auto-updates display |
| **`→`** (Right) | **Next Verse** | Advances one verse, auto-updates display |
| **`↑`** (Up) | **Previous Chapter** | Goes back one chapter |
| **`↓`** (Down) | **Next Chapter** | Advances one chapter |
| **`PgUp`** | **Previous Book** | Goes to previous book in Bible |
| **`PgDn`** | **Next Book** | Goes to next book in Bible |

### **Auto-Update Feature** 🔄
When display window is open, pressing `←` or `→` automatically updates what's shown on the projector!

## 🎵 **SONG VIEWER SHORTCUTS**

| Key | Action | Details |
|-----|--------|---------|
| **`←`** (Left) | **Previous Slide** | Goes to previous lyric slide |
| **`→`** (Right) | **Next Slide** | Goes to next lyric slide |
| **`Home`** | **First Slide** | Jumps to beginning of song |
| **`End`** | **Last Slide** | Jumps to end of song |
| **`Esc`** | **Close Viewer** | Returns to song list |

## 🎯 **Quick Workflow Examples**

### **Display a Bible Verse**
1. Select book, chapter, verse
2. Press **`Enter`** → verse appears on projector
3. Use **`→`** to go to next verse (auto-updates)
4. Press **`Esc`** when done

### **Navigate Through Bible**
1. **`PgDn`** → Next book
2. **`↓`** → Next chapter  
3. **`→`** → Next verse
4. Press **`Enter`** to display at any point

### **Display a Song**
1. Click song from list
2. Press **`→`** to advance slides
3. Press **`Esc`** when finished

## 💡 **Pro Tips**

- **Practice shortcuts before service** - Speed improves with muscle memory
- **Arrow keys are your friends** - Fastest way to navigate
- **Keep display window open** - Auto-update works only when window is open
- **Use P for preview** - Check verse before displaying to congregation

---

## 📱 **From Input Fields**

When typing in search boxes or input fields:
- **`Enter`** while in any input → **Display** (jumps straight to projector)
- Other shortcuts disabled to avoid conflicts while typing
