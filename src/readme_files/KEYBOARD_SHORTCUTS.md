# ⌨️ Keyboard Shortcuts Guide

## Bible Section Shortcuts

### Control Panel

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Enter** | Show Verse | Display the selected verse on projection screen |
| **Esc** | Clear Display | Remove verse from projection screen |
| **↑** | Previous Chapter | Navigate to previous chapter |
| **↓** | Next Chapter | Navigate to next chapter |
| **←** | Previous Verse | Go to previous verse |
| **→** | Next Verse | Go to next verse |

### Display Screen

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Esc** | Clear Display | Hide the displayed verse |

## Usage Tips

### Quick Verse Display
1. Use arrow keys to navigate to desired verse
2. Press **Enter** to show immediately
3. Press **Esc** to clear when done

### Rapid Navigation
- **↑↓** keys jump between chapters
- **←→** keys move through verses
- Navigation is instant

### While Typing
- **Enter** works even when typing in input fields
- Shortcuts disabled while typing (except Enter)
- This prevents accidental navigation

## Workflow Examples

### Example 1: Scripture Reading
```
1. Arrow Down to Chapter 3
2. Arrow Right to Verse 16
3. Press Enter to display
4. (Pastor reads verse)
5. Press Esc to clear
```

### Example 2: Multiple Verses
```
1. Select start verse with arrows
2. Manually set end verse in input
3. Press Enter to display range
4. Press Esc to clear
```

### Example 3: Quick Switch
```
1. Show John 3:16 (Enter)
2. Arrow Right to Verse 17
3. Enter to show John 3:17
4. Esc to clear
```

## Admin Features

### Dynamic Backgrounds

**Change Color:**
1. Admin Panel → Settings
2. Click preset or use color picker
3. Changes apply immediately

**Set Wallpaper:**
1. Admin Panel → Settings
2. Enter image URL in "Background Image/Wallpaper"
3. Press Tab or click away
4. Background updates on all screens

**Example URLs:**
```
https://images.unsplash.com/photo-1234567890
https://yourchurch.com/backgrounds/worship.jpg
```

## Pro Tips

### Speed Navigation
- Hold arrow keys for rapid movement
- Enter works from any state
- Esc always clears (safe to spam)

### Best Practices
1. **Pre-load chapter** before service starts
2. **Use arrow keys** for quick changes
3. **Keep display window open** on projector
4. **Practice shortcuts** before Sunday

### Troubleshooting
- **Shortcut not working?** Check if cursor is in input field
- **Display not updating?** Ensure display window is open
- **Wrong verse showing?** Press Esc and try again
